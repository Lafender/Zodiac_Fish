# Emojis Used
Below is a list of emojis used in the Folder Tree above. The file type each emoji represents is written in the right-most column.

| Emoji | Name           | File Type                                   |
| ----- | -------------- | ------------------------------------------- |
| 📁    | File Folder    | Directory                                   |
| 📄    | Page Facing Up | Other Text File (json, py, cvs, txt...)     |
| 🖼️   | Framed Picture | Image Files (.png, .jpg)                    |
| 🎬    | Clapper Board  | Video Files (.mp4... )                      |
| 🎶    | Musical Notes  | Audio Files (.mp3 or .ogg)                  |
| 💿    | Optical Disk   | GDScript that extends Node                  |
| 🏷️   | Label          | GDScript that extends Resource              |
| 📦    | Package        | Packed Binary File (.tscn or .tres)         |
| 🔧    | Wrench         | GDScript that extends RefCounted or nothing |
